Link to clone project: https://github.com/abhinav202/ClipBoard_Test.git

How to run from eclipse: 
1. Clone project from git
2. Open eclipse. Right click on the project
3. Run As -> Maven Clean
4. Run As -> Maven test

How to run from CLI:
1. Clone project from git
2. Open terminal
3. Run command: mvn clean test
